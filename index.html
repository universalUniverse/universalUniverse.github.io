OUR MISSION

This website exists to encourage curiosity. Designed with High schoolers in mind, we hope that this site will present information about the heavens in a manner that is easy to navigate and understand, in order to inspire the next generation of dreamers.

In the Navigation bar links containing information on the Solar System, Galaxy, and Universe can be found. In addition, you can find on this page a link to a list of external sources utilized to create this site.
